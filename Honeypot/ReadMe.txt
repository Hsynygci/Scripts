Simple honeypot implementation with python.

Requirements:
Apache2
php
and locage index.php into "www/html/"

Usage:
python hCore [host_ip]

Features:
- Open and listen ports for Siemens SCADA manual.
- Sample HMI interface web page login for Bruteforce attack detection.
- Proactive mod, insta whois/dig/tracepath for adversary IPs.
- Shodan detection, checking predefined shodan IPs on NIC's remote addr stack.
